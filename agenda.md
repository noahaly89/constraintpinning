# Research Agenda

Priority research questions include:

- How should constraints survive context compression and agent delegation?
- Which controls require deterministic validation rather than model judgment?
- How can constraint conflicts be detected before runtime?
- How should evidence be standardized across heterogeneous agent platforms?
- What formal guarantees can be made about contract inheritance?
- How should emergency exceptions remain safe, time-bound, and auditable?

The project welcomes empirical evaluations, formal models, implementation reports, and governance case studies.
